# 💼 Personal Portfolio

A modern and responsive portfolio website built to showcase my skills, featured projects, and experience as a MERN Stack Developer.

## 🌐 Live Demo

🔗 https://mirnatul.netlify.app/

## ✨ Features

- Responsive design for desktop, tablet, and mobile
- Modern and clean user interface
- Smooth animations and interactive effects
- Featured projects with live demo and GitHub links
- Skills categorized by technology stack
- About Me section
- Contact section
- Fast and optimized performance

## 🛠️ Tech Stack

### Frontend
- React.js
- Tailwind CSS
- JavaScript (ES6+)
- HTML5
- CSS3

### Libraries
- React Icons
- Lottie React
- React Router DOM
- SweetAlert2

### Deployment
- Netlify

## 📂 Featured Projects

### 🏕️ SummerSports
A summer camp management system with secure authentication, role-based dashboards, class management, and payment integration.

### 🚗 CarzzZone
A car marketplace where users can browse, add, update, and manage vehicle listings with a clean and responsive interface.

### ⚽ Squad
A sports team management platform that allows users to create teams, manage players, organize events, and collaborate efficiently.
